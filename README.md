# CS-Platform

A visual, interactive platform for learning Computer Science concepts — built with React, TypeScript, and D3.

## Features

- Interactive visualizations of CS concepts powered by D3
- Smooth animations with Framer Motion
- State management with Zustand
- Routing with React Router

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** (styling)
- **D3.js** (visualizations)
- **Framer Motion** (animations)
- **Zustand** (state management)
- **Vitest** (testing)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```
